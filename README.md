EchoPlex

EchoPlex is a HISE effect project focused on a tape style delay processor.
The main interface controls free delay time synced delay time repeats drift input level output level and wet mix.
The project also includes animated LED style metering and a custom front panel.

The main delay network is built around a Faust based Echoplex processor with controls for delay time feedback drift flutter wow record level playback volume bass gain and treble gain.
Tempo sync is handled through a dedicated control node.

The project also includes additional effect networks and source code for greyhole and vitalRev based processing.
Presets images DSP network files and generated build files are stored in the project tree.

Open the project in HISE to edit the interface script code Faust DSP files and generated scriptnode sources.
